# lecture6_examples
Example of using a URDF file with a scara robot

# Build example
```bash
cd ~/workspace/examples_ws/
```
```bash
colcon build
```
# Run exanple
```bash
cd ~/workspace/examples_ws/
```
```bash
cd source install/setup.bash
```
```bash
ros2 launch scara_urdf_example scara.launch.py
```
